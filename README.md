## Getting Started

### Install dependencies

```bash
pip install jupyter notebook
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cpu
```
### Run the notebooks
```bash
jupyter notebook
````
Then open any chapter-*.ipynb file and start exploring.
